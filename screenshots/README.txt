This directory will contain screenshots of the Mental Health Chat Assistant application.

To add screenshots:
1. Take a screenshot of your application
2. Name it appropriately (e.g., "chat_interface.png", "settings_page.png")
3. Place it in this directory
4. Update the README.md file to reference these screenshots

Recommended screenshots to add:
- chat_interface.png: Main chat interface with example conversation
- settings_page.png: Settings page showing configuration options
- resources_page.png: Mental health resources page

These screenshots will help users understand the application's features and interface.
